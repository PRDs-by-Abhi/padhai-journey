# PadhAI UPSC — User Journey to Premium

This folder is the source for GitHub Pages. The main document is **index.html** (User Journey flowcharts, Message Directory, Template Library, PRD for Devs).

## Updating the live site

After you change `user-journey-flowchart.html` in the project root, copy it here so the site updates:

```bash
cp user-journey-flowchart.html docs/index.html
git add docs/index.html && git commit -m "Update journey/PRD" && git push
```
